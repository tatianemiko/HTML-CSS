Principais Formatações

Negrito/ Destaque

Nesta frase, temos um termo em negrito, usando a tag B (não semântico, ou seja, sem significado apenas a forma)

Nesta frase, temos um termo em destaque usando a tag STRONG (semântico, ou seja, ela tem um significado "ela é uma palavra/ frase FORTE")

Itálico/ Ênfase

Nesta frase, temos um termo em itálico usando a tag i (não semântico)

Nesta frase, temos um termo em itálico usando o comando CTRL+SFT+P > abb > ENTER > I > ENTER com a frase "um termo em itálico" selecionado

Nesta frase, temos um termo em ênfase usando a tag EN (semântico)

Texto Marcado

Podemos criar também um texto marcado com a tag MARK

E no outro parágrafo temos outro texto marcado no final

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Texto Deletado

Podemos marcar um texto como excluido, para indicar que ele deve ser lido mas não considerado

Texto Inserido

Podemos marcar um texto como inserido, para dar uma ênfase/ indicar que ele foi inserido, usando a tag INS (semântico)

Podemos marcar um texto inserido, para indicar (forma) com a tag U

Texto Subescrito

Para inserir caracteres do tipo x2+3, utilizando a tag SUP

Para inserir caracteres como H2O, utilizando a tag SUB
